# Environment Variables

### TANKA_JB_PATH

**Description**: Path to the `jb` tool executable
**Default**: `$PATH/jb`

### TANKA_KUBECTL_PATH

**Description**: Path to the `kubectl` tool executable  
**Default**: `$PATH/kubectl`

### TANKA_KUBECTL_TRACE

**Description**: Print all calls to `kubectl`  
**Default**: `false`
